# DMs Toolbox Offline — GitHub Actions Builders

This repo contains:
- **Windows Electron Offline** builder workflow
- **Android Capacitor** builder workflow

## Run a workflow
Go to **Actions** → pick a workflow → **Run workflow**.

## If you don't see workflows
Make sure:
- workflow files exist in `.github/workflows/`
- they are on the **default branch** (usually `main`)
- Actions are enabled in repo Settings → Actions → General
